Bem-vindo ao meu repositório de estudos e projetos práticos! 🚀

Este espaço é dedicado a armazenar e documentar a minha evolução técnica ao longo do curso de **Desenvolvimento de Sistemas no SENAI**. Aqui, você encontrará desde os meus primeiros passos na lógica de programação até aplicações back-end mais estruturadas e complexas.

## 🗂️ Estrutura do Repositório

O repositório está dividido em duas pastas principais, refletindo a minha jornada de aprendizado:

### 1. ☕ `Java/` (Projetos Avançados)
Nesta pasta estão os projetos mais robustos e complexos, onde aplico conceitos de Programação Orientada a Objetos (POO) e persistência de dados.
*   **Integração com Banco de Dados:** Aplicações conectadas ao banco de dados relacional **MySQL**.
*   **Gerenciamento de Dependências:** Uso do **Maven** para gerenciar as bibliotecas e o ciclo de vida dos projetos (ex: projeto `ControleDeAcesso`).
*   **Padrões de Projeto:** Estruturação de código visando boas práticas, manutenção e escalabilidade.

### 2. 📝 `Portugol/` (Lógica e Algoritmos)
Onde tudo começou! Esta pasta contém exercícios focados no desenvolvimento da lógica de programação pura.
*   Criação de algoritmos estruturados.
*   Uso de laços de repetição, condicionais e estruturas de dados básicas (como matrizes e vetores).

## 🛠️ Tecnologias e Ferramentas Utilizadas

*   **Linguagens:** Java, Portugol
*   **Banco de Dados:** MySQL
*   **Ferramentas:** Maven, Git, GitHub
*   **IDE:** IntelliJ IDEA

## 🚀 Como executar os projetos Java

Para clonar e executar os projetos de backend na sua máquina, siga os passos abaixo:

1. Clone este repositório:
   ```bash
   git clone https://github.com/viniciusalmeideavila/SENAI.git

2. Importe o projeto desejado na sua IDE de preferência.
3. Certifique-se de que o Maven baixou todas as dependências listadas no arquivo pom.xml.
4. Configure as credenciais do banco de dados MySQL no arquivo de configuração do projeto.
5. Execute a classe principal (Main) do projeto.
