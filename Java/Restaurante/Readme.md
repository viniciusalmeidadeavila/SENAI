🍔 Blend Lunch - Sistema de Gestão de Restaurante
Bem-vindo ao Blend Lunch, um sistema de console desenvolvido em Java para gerenciar pedidos e cadastros de um restaurante de forma simples e eficiente.

# 🚀 Funcionalidades
* O sistema está dividido em dois grandes pilares

# 1. Realizar Pedidos
* Permite a busca de Pratos ou Bebidas no cardápio.

* Validação em tempo real: o sistema verifica se o item solicitado existe antes de confirmar a escolha.

# 2. Área de Cadastros (Administrativo)
* Produtos: Cadastro de novos pratos (com ingredientes) e bebidas (com volume em ML).

* Funcionários: Registro de equipe com distinção de cargos:

* Garçom

* Cozinheiro

* Gerente (com definição de senha de acesso)
